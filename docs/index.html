GRAPH

HNSW with α-RNG Pruning

Hierarchical Navigable Small World with Vamana-style α=1.2 neighbor selection. M=16, ef_c=200. DashMap concurrent writes, ArcSwap wait-free reads.

QUANTIZATION

RaBitQ Multi-Bit

Randomized Binary Quantization (SIGMOD 2024). Fast Hadamard rotation preserves inner products. 4 bit-planes with NEON vtstq_u32 branchless masked sums.

SILICON

Apple Silicon Native

Metal compute shaders for batch L2/cosine (1.4× at 64K). UMA zero-copy with StorageModeShared. ARM64 prfm speculative prefetch. Hot/cold tier HNSW.

TOPOLOGY

Persistent Homology

H₀/H₁ Betti numbers via ego-graph construction. β₁ captures "loop density" around candidates — penalizes hub nodes, rewards locally connected clusters.
