  • binary: Option<Vec<u8>> — Optional binary representation
  • neighbors: Vec<Vec<NodeId>> — Multi-layer adjacency list
  • level: usize — Layer assignment in hierarchy
  • metadata: MetaData — Associated metadata

STORAGE

HnswRaBitQConfig

Configuration structure for index creation.

  • dimensions: usize — Vector dimension
  • max_neighbors: usize — HNSW M parameter
  • ef_construction: usize — HNSW ef_c parameter
  • rabitq_bits: usize — Quantization bits (1 or 4)
  • rerank_top: usize — Top-K before reranking
  • metric: HnswDistanceMetric — L2 or Cosine
  • seed: u64 — RaBitQ random seed
STORAGE

RaBitQSearchResult

Result from hybrid search.

  • node_id: NodeId — Matched node
  • estimated_dist: f32 — RaBitQ estimate
  • precise_dist: Option<f32> — Exact distance (if computed)
